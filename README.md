## Installation
Get driver to use the Firefox browser
'''
sudo apt install firefox-geckodriver
'''
Install all necessary python modules
'''
mkvirtualenv INSTAPY -p python3  # (OPTIONAL)
pip install instapy
'''

## Run Simulation 
'''
workon INSTAPY      # Turn on virtual env (OPTIONAL)
python start.py 
'''
